# Terminal UI

Functions for the terminal user interface that allows for the management of the server-side application.

The package [simple_term_menu](https://github.com/IngoMeyer441/simple-term-menu/tree/develop) is used to create the terminal user interface.

## Functions

### main_menu

The function that renders the main settings menu.

#### Example

```python
from term_ui import main_tui

main_tui.main_menu()
```
